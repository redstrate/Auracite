# Auracite

Archive your FFXIV character in a portable, generic format. This includes
data in machine-readable JSON (and can be imported by other programs
like [Kawari](https://github.com/redstrate/Kawari).)

## Usage

It works by running in Dalamud, and the plugin is currently available
[in my personal Dalamud repository](https://github.com/redstrate/DalamudPlugins).
